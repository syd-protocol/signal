# /SIGNAL — INSTAGRAM CAROUSEL GUIDE
## How to post each chapter daily

---

## HOW THIS WORKS

Each day you post one chapter as a carousel.
A carousel is a set of slides — between 3 and 6 images uploaded together as one Instagram post.
Each slide is one exported PNG from Figma.

Your Figma file is already set up:
- Frame: 1080 x 1350
- Text layer: X 80, Y 180, W 920, H 980
- Font: Share Tech Mono, size 32, cyan 4FC3F7, line height 150%, letter spacing 2%
- Background: locked, never touch it

**Your daily workflow is:**
1. Open Figma
2. Duplicate your existing frame for each slide
3. Paste the text for each slide from this guide
4. Export all slides as PNG
5. Upload to Instagram as a carousel in order
6. Copy and paste the caption from this guide

That is it. Nothing else to decide.

---

## TWO TEXT STYLES

Every chapter has two types of text. You set these up as separate text layers in Figma.

**SYSTEM TEXT** — used for all `[ BRACKET ]` blocks
- Font: Share Tech Mono
- Size: 32
- Colour: 4FC3F7 (cyan)
- Letter spacing: 2%
- Line height: 150%

**PROSE TEXT** — used for all narrative paragraphs
- Font: DM Sans
- Size: 30
- Colour: E2E8F0 (near white)
- Letter spacing: 0%
- Line height: 160%

When a slide has both types, create two separate text layers stacked vertically within the text zone. System text first if the System speaks first, prose first if the narrative opens.

---

## FIGMA STEP-BY-STEP FOR EACH NEW POST

1. Open your Figma file
2. In the layers panel, right-click your last frame → **Duplicate**
3. Rename the new frame to match the slide (e.g. `T-002 Slide 1`)
4. Double-click the text layer inside the new frame
5. Select all text (`Ctrl/Cmd + A`) and delete it
6. Paste the exact text from this guide for that slide
7. Check the text fits — if it runs over the bottom of the text zone, reduce font size to 28
8. Click the frame name in the layers panel
9. Scroll to Export in the right panel → PNG → 1x → Export
10. Repeat for each slide in the chapter
11. Upload all slides to Instagram as one carousel post in order
